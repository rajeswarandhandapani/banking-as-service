# Project Guidelines

- **Language**: Java
- **Java Version**: 21

## Guidelines

1. Use Java 21 syntax and features.
2. All common constants must be placed in the `common-lib` module.
3. Follow standard Java naming conventions.
4. Encapsulate reusable logic in `common-lib` when appropriate.
5. Make sure the changes are compatible and consistent with the existing codebase.
6. Create changes step by step, allowing for review and feedback at each stage.
7. make sure to use builders to construct saga events.
